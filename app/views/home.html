The first step to keep online information safe is to know where this information is and to manage it properly.

Before your start

  1. It is crucial for some features (e.g. Facebook) that you modify the URL. Indeed, the application usually boots on your localhost address IP which is not what we want. To make sure the URL does not cause problem, change the IP address into 'localhost'. For example, if the URL is 127.0.0.1:9000/, change it into localhost:9000/.
  2. You will find in the documentation of the application various information that could be useful during the experience (e.g. goal of the application, ethical considerations, features of the application, potential source of errors...).
  3. If you wish to be guided while using the application, click this icon while testing the application.
