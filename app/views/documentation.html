Goals of the SearchGuardian

This is an application developed in order to help internet users to protect their online privacy and be aware of information about them displayed without their knowledge and consent. None of the information input is saved on the server side, every information input by the user and retrieved through the applications should be deleted as long as the user logs out properly

Context of the project

This web application was developed as part of my MSc project This is only a prototype of the SearchGuardian as the Web is so vast it will be hard to cover it all. The web application is still in development and the current project is available at this URL: https://github.com/jtran1403/SearchGuardian

How does it work

The web application was developed using Yeoman (Grunt and Bower), AngularJS and Karma.
